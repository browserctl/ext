# Browserctl Extensions

This directory contains the browserctl Chrome extension.

## Contents

- [chromium/](chromium/) - Chrome extension (Manifest V3)

## Quick Start

1. Install the extension: see [chromium/INSTALL.md](chromium/INSTALL.md)
2. Start the service: `sudo systemctl start browserctl-svc`
3. Connect from Chrome toolbar
